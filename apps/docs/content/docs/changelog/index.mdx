## v0.1.0

*Initial Release*

The first public release of FlareDeck — a desktop control panel for Cloudflare Tunnel management.

### Features

- **Profile Management** — Create, edit, and delete named tunnel configurations
- **Visual Ingress Editor** — Add and reorder hostname-to-service mapping rules through a graphical interface
- **DNS Routing** — Configure DNS routes for tunnels directly from the app
- **Live Monitoring** — Real-time log streaming from `cloudflared` with color-coded output levels
- **Tunnel Lifecycle** — Start, stop, and monitor tunnel processes with status indicators
- **Raw YAML Editor** — Full access to `cloudflared` configuration files with syntax highlighting
- **WSL Support** — Automatic IP rewriting for services running inside Windows Subsystem for Linux
- **Theme Support** — Light, dark, and system-following theme modes
- **Auto-detection** — Automatic discovery of `cloudflared` binary and existing tunnel configurations

### Platform Support

- Windows 10 (version 1803+) and Windows 11
- macOS and Linux support planned for future releases
